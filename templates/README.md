# Document Templates

This directory contains document templates organized by maturity level.

## Structure

- **[taxonomy/](taxonomy/)** - Taxonomy and classification templates
- **[mvp/](mvp/)** - Minimum Viable Product templates  
- **[production/](production/)** - Production-grade templates
- **[enterprise/](enterprise/)** - Enterprise-grade templates

## Maturity Levels

### Taxonomy
Foundation-level templates for establishing terminology, categorization, and classification systems. Use these when starting a new project or establishing organizational structure.

### MVP (Minimum Viable Product)
Streamlined templates for rapid prototyping and proof of concept. Focus on essential documentation without overhead.

### Production
Full-featured templates suitable for production deployments. Includes comprehensive documentation for stable, tested systems.

### Enterprise
Enterprise-grade templates with compliance, audit trails, and governance requirements. Suitable for large organizations with regulatory needs.

## Usage

1. Choose the appropriate maturity level for your project phase
2. Select specific templates from the chosen category
3. Customize templates to fit your project requirements
4. Maintain version control for all documentation
