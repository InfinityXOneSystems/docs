# Coding & Software Development Templates

Document templates for software development lifecycle and engineering practices.

## Categories

### Architecture
- System architecture documentation
- Design decision records (ADRs)
- Technical specifications

### Requirements
- User stories
- Feature specifications
- Acceptance criteria

### API Documentation
- API specifications (OpenAPI/Swagger)
- Endpoint documentation
- Integration guides

### Testing
- Test plans
- Test case templates
- QA documentation

### Deployment
- Deployment guides
- Runbooks
- Configuration documentation

### Security
- Security requirements
- Threat modeling
- Vulnerability assessment

### Code Quality
- Code review checklists
- Coding standards
- Style guides

## Available Templates

*Templates will be added as the library grows.*

## Contributing

To contribute a new template:
1. Follow the existing template format
2. Include usage instructions
3. Provide examples where applicable
4. Submit a pull request
