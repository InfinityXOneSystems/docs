# Taxonomy Templates

Foundation-level templates for establishing terminology, categorization, and classification systems.

## Purpose

Taxonomy templates help establish:
- Consistent terminology across projects
- Classification hierarchies
- Categorization standards
- Organizational structure definitions

## Available Templates

*Templates will be added as the library grows.*

## Usage Guidelines

1. Start with taxonomy templates when beginning a new project
2. Establish naming conventions early
3. Document all classification decisions
4. Review and update taxonomy as the project evolves
